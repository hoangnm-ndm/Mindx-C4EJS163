# CSS

CSS = Cascading Style Sheets (Ngôn ngữ tạo phong cách dạng xếp chồng)

## Cách nhúng CSS vào HTML

### Inline CSS

- Có độ ưu tiên cao nhất.
- Không tái sử dụng được code CSS.

### Internal CSS

- Có độ ưu tiên thấp hơn Inline CSS.
- Có thể tái sử dụng được code.
- Tuy nhiên sử dụng trong file HTML -> không tối ưu.

## External CSS

- Có độ ưu tiên thấp hơn Inline CSS.
- Có thể tái sử dụng được code.

> Lưu ý: Trong 2 cách viết internal và external CSS thì không cách nào mạnh hơn, mà cách nào viết sau sẽ xếp chồng lên cách viết trước (theo nguyên tắc CSS).
