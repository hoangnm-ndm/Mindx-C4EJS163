# Phương thức `reduce2()`

Hãy mở rộng `Array.prototype` bằng cách thêm một phương thức mới là `reduce2(callback, initialValue)` có chức năng tương tự như phương thức `reduce()` nguyên bản của JavaScript.

**Cụ thể:**

- Phương thức này nhận một hàm `callback` làm tham số đầu tiên. Hàm này sẽ được gọi cho từng phần tử trong mảng.
- Hàm `callback` nhận vào 4 tham số: giá trị tích lũy (hoặc giá trị khởi tạo), giá trị hiện tại, chỉ số của phần tử và chính mảng đó.
- Tham số thứ hai là `initialValue` (không bắt buộc). Nếu được cung cấp, giá trị này sẽ được sử dụng làm giá trị khởi tạo cho `accumulator`. Nếu không, giá trị đầu tiên của mảng sẽ được sử dụng.
- Phương thức sẽ trả về giá trị tích lũy cuối cùng sau khi lặp qua tất cả các phần tử trong mảng.
- Phương thức chỉ lặp qua các phần tử có chỉ số từ `0` đến `length` ban đầu của mảng (nếu mảng thay đổi trong quá trình lặp, số lần lặp không bị thay đổi).
- Phương thức sẽ bỏ qua các phần tử trống (empty elements) trong mảng khi gọi `callback`.
- Nếu mảng rỗng và không có `initialValue`, phương thức sẽ thông báo một lỗi.
- Nếu mảng chỉ chứa một phần tử và không có `initialValue`, phương thức sẽ trả về phần tử đó.
- Nếu mảng rỗng và có `initialValue`, phương thức sẽ trả về `initialValue`.

**Yêu cầu:** Không được sử dụng phương thức `reduce()` nguyên bản của JavaScript.

```javascript
Array.prototype.reduce2 = function (callback, initialValue) {};

// Sample 1
const arr1 = [1, 2, 3, 4, 5];

const result1 = arr1.reduce2((accumulator, value) => {
  return accumulator + value;
}, 0);

console.log(result1); // 15

// Sample 2
const arr2 = [];

const result2 = arr2.reduce2((accumulator, value) => {
  return accumulator + value;
});

console.log(result2); // Error: Reduce of empty array with no initial value

// Sample 3
const arr3 = [];

const result3 = arr3.reduce2((accumulator, value) => {
  return accumulator + value;
}, 0);

console.log(result3); // 0
```
