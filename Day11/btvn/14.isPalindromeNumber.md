# Viết hàm `checkSymmetricalNumber(n)` để kiểm tra số có đối xứng hay không?

- Số đối xứng là số mà đọc từ trái sang phải hay từ phải sang trái đều giống nhau.
- Ví dụ: 121, 12321, 34543, 123454321 là số đối xứng.
- Nếu số nhập vào chỉ có 1 chữ số, báo là `số có 1 chữ số, không kiểm tra được`.
- Nếu dữ liệu nhập vào không hợp lệ, báo là `Dữ liệu không hợp lệ`.

```javascript
// Input:
function checkSymmetricalNumber(n) {
  // Logic bài toán
}

// Output:
checkSymmetricalNumber(121); //true
checkSymmetricalNumber(123454321); //true
checkSymmetricalNumber("123454321"); // Dữ liệu không hợp lệ
```
