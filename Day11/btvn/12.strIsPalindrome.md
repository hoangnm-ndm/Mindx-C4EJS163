# Viết hàm `checkSymmetricalStr(str)` để kiểm tra chuỗi có đối xứng hay không?

- Chuỗi đối xứng là chuỗi mà đọc từ trái sang phải hay từ phải sang trái đều giống nhau.
- Ví dụ: "abba", "racecar", "madam" là chuỗi đối xứng.
- Nếu chuỗi nhập vào chỉ có 1 ký tự, báo là `chuỗi có 1 ký tự, không kiểm tra được`.
- Nếu dữ liệu nhập vào không hợp lệ, báo là `Dữ liệu không hợp lệ`.
- Nếu chuỗi là đối xứng trả về "true", ngược lại trả về "false".

```javascript
// Input:
function checkSymmetricalStr(str) {
  // Xử lý logic
}

// Output:
checkSymmetricalStr("abba"); // true
checkSymmetricalStr("abcd"); // false
```
