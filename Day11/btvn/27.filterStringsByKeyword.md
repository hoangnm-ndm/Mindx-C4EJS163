# Lọc các chuỗi chứa từ khóa

## Mô tả

Viết hàm `filterStringsByKeyword()` để lọc các chuỗi trong mảng chứa từ khóa cho trước.

- Nhận vào một mảng các chuỗi và một từ khóa.
- Trả về một mảng mới chỉ chứa các chuỗi có chứa từ khóa đó.

```javascript
const arr = ["apple", "banana", "cherry", "date"];
const keyword = "an";
const result = filterStringsByKeyword(arr, keyword);
console.log(result); // ["banana"]
```
