**Bài 1:** Viết hàm `formatDate(date)` nhận vào một đối tượng `Date` và trả về chuỗi định dạng "DD/MM/YYYY".

// Ví dụ sử dụng:

**Bài 2:** Viết hàm `addDays(date, days)` nhận vào một đối tượng `Date` và số ngày, trả về một đối tượng `Date` mới sau khi đã cộng thêm số ngày đó.

```javascript

// Ví dụ sử dụng:
const date = new Date(2023, 0, 15); // 15 January 2023

```

**Bài 3:** Viết hàm `daysBetween(date1, date2)` nhận vào hai đối tượng `Date` và trả về số ngày giữa hai ngày đó.

- Nếu `date1` trước `date2`, trả về số ngày dương.
- Nếu `date1` sau `date2`, trả về số ngày âm.

```javascript
function daysBetween(date1, date2) {}
// Ví dụ sử dụng:
const date1 = new Date(2023, 0, 15); // 15 January 2023
const date2 = new Date(2023, 0, 25); // 25 January 2023
console.log(daysBetween(date1, date2)); // 10
```
